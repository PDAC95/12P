How to Test

Desktop View (≥768px):

  1. Open the application at http://localhost:4200
  2. Navigate to the Properties page
  3. Click the "Map" view toggle button
  4. Verify the map displays with sidebar controls
  5. Test filtering by selecting locations and adjusting radius

Mobile View (<768px):

  1. Open browser DevTools (F12)
  2. Toggle device toolbar (Ctrl+Shift+M)
  3. Select a mobile device preset (e.g., iPhone 12 Pro)
  4. Navigate to Properties page and select "Map" view
  5. Verify map is hidden and search controls are visible
  6. Test each control:
    • Click "Use My Location" button
    • Enter an address and search
    • Select different cities
    • Click popular location tags
    • Adjust radius with slider and quick buttons
  7. Verify filtered properties appear below controls
